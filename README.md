# 🌌 Overview

**SmartDefectAI** is a production-grade AI-powered industrial defect detection platform built using **Deep Learning, FastAPI, React, Docker, and CI/CD pipelines**.

The platform classifies industrial steel surface defects in real-time using a trained deep learning model and serves predictions through a modern full-stack AI web application.

This project combines:

* 🧠 Deep Learning Engineering
* ⚡ Full-Stack AI Development
* ☁️ Cloud Deployment
* 🐳 Docker Containerization
* 🔄 CI/CD Automation
* 📊 Real-Time Inference APIs
* 🎨 Futuristic AI Frontend Design

# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Ashishh13/SmartDefectAI.git
cd SmartDefectAI
```

---

## 2️⃣ Backend Setup

```bash
python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt
```

Run backend:

```bash
uvicorn backend.app.main:app --reload
```

---

## 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# 🐳 Docker Deployment
