# SmartDefectAI

AI-powered industrial surface defect detection platform using Deep Learning, FastAPI, React, Docker, and CI/CD.

![Dashboard](docs/screenshots/dashboard.png)

## Features

- Real-time defect detection
- Deep Learning classification
- FastAPI inference backend
- React frontend dashboard
- Dockerized deployment
- CI/CD with GitHub Actions
- Cloud deployment with Render + Vercel

## Tech Stack

### AI/ML
- PyTorch
- torchvision
- scikit-learn

### Backend
- FastAPI
- Uvicorn

### Frontend
- React
- Vite
- Axios
- Framer Motion

### DevOps
- Docker
- GitHub Actions
- Render
- Vercel
